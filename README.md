# Hi, I'm shennoyi, a passionate about transhumanism. 👋

我是一名信息工程背景的开发者，关注 **AI、认知建模与复杂系统**，目前主要在做和「人如何思考、如何被理解」有关的事情。

---

## 🧠 我在做什么

- 🧩 构建 **以“概念 / 观念”为核心的认知分析系统**  
  试图从对话与行为中，抽象出稳定、可迁移的认知结构，而不止是标签或性格分类。

- 🤖 探索 **LLM 的结构性局限**  
  对 token 驱动的生成范式保持怀疑，正在尝试以 *概念激活网络 / 多层结构* 的视角，思考更接近人类认知的模型形态。

- 🚀 创业向项目：**Metalks**  
  一个面向年轻人的 AI 认知分析与深度对话平台，目标不是“陪伴”，而是**帮助人理解自己在想什么、为什么会这样想**。
